## Beginner

####  What is HTML? 

HTML(Hyper Text Markup Language) is a standard language used to create weg pages. It structures the content of web pages using elements and tags that browser can interpret. 
It is a markup language, not a programming language. This means it annotates text to define how it is structured and displayed by web pages. 
It is a static language, it doesn't inherently provide interactive features but can be combined with CSS for styling and JS for interactivity.
